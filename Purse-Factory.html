Purse Factory

About

This is a generative design which I coded in p5.js. I created a purse factory which generates purses randomly. Clicking on the "Click to Start" button will generate 6 purses. Everytime the button is clicked and the code is run, 6 new purses will be randomly generated. Meaning, each purse will have a randomly selected color, brand, and charm.
